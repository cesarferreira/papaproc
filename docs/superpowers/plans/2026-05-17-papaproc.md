# papaproc Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a dependency-aware local process orchestrator with TUI, readiness probes, diagnostics, and snapshots.

**Architecture:** Single Rust binary crate with focused modules for config, graph, supervisor, readiness, diagnostics, TUI, and snapshots. Process supervision updates shared state; the TUI renders state and dispatches actions.

**Tech Stack:** Rust, Tokio, Ratatui, Crossterm, Clap, Serde YAML, Reqwest, Regex, Anyhow.
